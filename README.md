## Diabetes Prediction App

A machine learning web application to predict diabetes risk using medical features.

### Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- XGBoost
- Streamlit

### Features
- User input based prediction
- Displays diabetes risk probability
- Live deployed Streamlit app

### Live App
https://diabetes-prediction-app-pro.streamlit.app/

⚠️ Disclaimer: This application is for educational purposes only and not a medical diagnosis.
