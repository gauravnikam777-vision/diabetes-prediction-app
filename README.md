## Diabetes Prediction App

A machine learning web application to predict diabetes risk using medical features.

### Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- XGBoost
- Streamlit

### Features
- User input based prediction
- Displays diabetes risk probability
- Live deployed Streamlit app
